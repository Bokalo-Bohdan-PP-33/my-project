# Проєкт: Комплексний аналіз ризиків та QA контроль


\# Financial Risk Analysis MVP

\### Опис MVP

Система для ідентифікації та аналізу фінансових ризиків у межах лабораторної роботи. Дозволяє розраховувати ймовірність настання ризиків та їхній вплив.



\### Стек технологій

\* ReactNative (Frontend)

\* Git



\### Інструкція з запуску

1\. Схилюйте репозиторій: \\`git clone <url>\\`

2\. Перейдіть у папку: \\`cd my-project\\`

3\. Запустіть проєкт: \\`npm install \&\& npm start\\` (якщо додано конфігурацію)
